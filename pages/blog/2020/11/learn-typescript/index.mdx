# I Learn TypeScript

Мои первые функции, типизированные с помощью TypeScript :)

```ts
function getPropertyValue<T, K extends keyof T>(item: T, key: K): T[K] {
  return item[key];
}

function getItemBy<T, K extends keyof T>(
  items: T[],
  key: K,
  value: T[K]
): T | undefined {
  return items.find((item: T) => item[key] === value);
}
```
